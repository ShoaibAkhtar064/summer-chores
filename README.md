# Summer Chores

This project demonstrates callback functions and callback hell using JavaScript and Node.js.

## Features
- Uses asynchronous functions with setTimeout
- Demonstrates callback chaining
- Randomly determines if the person falls asleep

callbackVersion.js
Demonstrates callback functions and callback hell by completing a series of summer chores in a specific order.

promiseVersion.js
Demonstrates how to replace callback hell with Promises and promise chaining.


## Technologies Used

JavaScript
Node.js
Callbacks
Promises

## Run Project

'bash
Run the callback version:
node callbackVersion.js

Run the promise version:
node promiseVersion.js
